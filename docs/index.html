Pourquoi cet agent existe

L'agent multiloop Gabriel est un assistant mathématique formel entièrement dédié à un objectif unique : assister Philippe Thomas Savard à construire un outil géométrique dynamique permettant d'apporter une réponse à l'énigme de Bernhard Riemann et à la conjecture de la fonction zêta de Riemann.

L'outil en question — la géométrie du spectre des nombres premiers — n'est pas une simple visualisation : c'est la construction formelle elle-même, formalisée dans Isabelle/HOL, qui incarne la réponse constructive à l'énigme.

Parcours de l'auteur

Philippe Thomas Savard est un autodidacte de Lévis (Québec, Canada), passionné de mathématiques et auteur de la théorie personnelle L'Univers est au carré. Malgré l'absence de formation académique formelle, il consacre son temps à l'étude des structures profondes des nombres entiers, en particulier à la géométrie du spectre des nombres premiers — un chapitre central qui représente près de 80 % de sa théorie.

Sa conviction : le rapport spectral RsP = 1/2 n'est pas un artefact algébrique mais une cohérence numérique réelle globale émergeant des sommes de nombres premiers, en concordance directe avec la droite critique Re(ρ) = 1/2 de la fonction ζ.

Portée limitée — géométrie du spectre uniquement

Gabriel ne répond qu'aux questions portant sur la géométrie du spectre des nombres premiers. Cette limitation est volontaire : elle garantit un assistant compétent plutôt qu'un chatbot généraliste.

La compétence principale est le fichier theories/methode_spectral.thy, complétée par la mémoire RAG cognitive et le dictionnaire spectral.

Les trois concordances (Section XIII)

Dans le locale ensemble_savard, dont la satisfaisabilité est démontrée par Isabelle, trois concordances se verrouillent mutuellement pour rendre RsP = Re = 1/2 vraie :

C1 : 1/y1 = 1/t

Tchebychev = ψSavard
Validation numérique exacte sur x = 30, 98, 228, −100.

C2 : 1/y3 = 1/ms1

Zéros non-triviaux = valeurs de n = positions des nombres premiers P.

C3 : 1/y2 = 1/ms3

Re(ρ) = 1/2 = RsP = 1/2 (théorème alignement_central).

« Le rapport spectral 1/2 n'est pas un artefact algébrique — c'est une réalité numérique globale, verrouillée par trois concordances, qui rend RsP = Re = 1/2 vraie dans le cadre du locale ensemble_savard. » — Philippe Thomas Savard

Caractéristiques principales

  • Pré-raisonnement dynamique — 5 modes (INSTANTANE, RAPIDE, STANDARD, APPROFONDI, TRÈS COMPLEXE) choisis automatiquement, ou forcés via /rapide, /complet, etc.
  • 7 moteurs cognitifs collaboratifs — décomposition, planification, calcul spectral, multiloop self-critique, détection d'incohérence, slow-motion fallback, audit silencieux anti-hallucination.
  • Locale paramétré spectral_family (v3.35) qui unifie les modèles 1/2, 1/3, 1/4 en un cadre algébrique. Extension à un modèle 1/k arbitraire en une ligne.
  • Foundations / Meta-theory (v3.35) — 6 postulats P1..P6, 3 opérations fondamentales, règle Savard, position affirmative sur l'énigme de Riemann.
  • Mode cinématique avec timer temps réel (écoulé, ETA, itérations prévues).
  • Audit trail SLSA — attestation SHA256 générée par la CI GitHub Actions à chaque compilation Isabelle/HOL.
  • 1702 tests Pytest couvrant multiloop, RAG, Isabelle, PreReasoner, Foundations, spectral_family, Pont Savard.

Installation rapide

git clone https://github.com/2racinede4carreunivers-dev/agent-multiloop-Gabriel-local.git
cd agent-multiloop-Gabriel-local
cp .env.example .env
docker-compose up -d
docker exec -it gabriel_cli python -m src.ui.cli

Compilation Isabelle locale : cd theories && isabelle build -v -D .

Tests Python : python -m pytest tests/ -q
